ğŸš€ Getting Started
------------------

Clone this repository to your local machine.

```bash
git clone https://github.com/your-username/getir-clone.git
```

Navigate to the project directory.

```bash
cd getir-clone
```

Install dependencies.

```bash
npm install
```

To run the project in development mode, execute:

```bash
npm run dev
```
